## Getting Started
1. **Clone repository ini**
```
git clone https://gitlab.informatika.org/tubes-rpl-kelompok-8/gymbro-management-system.git
```
2. **Install node modules**
```
npm install
```
3. **Jalankan development server:**

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
